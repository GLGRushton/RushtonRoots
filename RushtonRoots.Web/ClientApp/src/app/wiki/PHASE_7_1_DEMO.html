1. WikiIndexComponent app-wiki-index

Main wiki navigation and article listing component with comprehensive filtering and sorting.

Features

  • Grid and list view modes with responsive layout
  • Article search with real-time debounced filtering
  • Category filtering with icons and colors
  • Status filtering (Published, Draft, Archived)
  • Multiple sort options (title, date, views)
  • Article cards with metadata display
  • View count and version tracking
  • Locked article indicators
  • Empty state handling
  • Mobile-responsive design

Usage

<!-- In Razor view -->
<app-wiki-index></app-wiki-index>

Sample Data Included:

  • 6 articles with different statuses
  • 5 categories: Family History, Research Tips, Record Types, How-To Guides, Best Practices
  • Locked article example
  • Version tracking demonstration

2. WikiArticleComponent app-wiki-article

Displays wiki articles with rendered markdown content and auto-generated table of contents.

Features

  • Markdown rendering using marked library (v14.1.5)
  • Auto-generated hierarchical table of contents
  • Sticky TOC sidebar with smooth scrolling
  • Article metadata (author, date, version, views)
  • Breadcrumb navigation
  • Print-friendly view
  • Version history access
  • Comprehensive markdown styling (headings, lists, tables, code blocks, blockquotes, images)
  • Responsive layout with mobile TOC handling

Usage

<!-- By article ID -->
<app-wiki-article [articleId]="1"></app-wiki-article>

<!-- By article slug -->
<app-wiki-article [articleSlug]="'getting-started-genealogy'"></app-wiki-article>

Markdown Elements Styled:

  • Headings (h1-h6) with visual hierarchy
  • Links in brand color (#2e7d32)
  • Lists (ordered and unordered)
  • Blockquotes with left border accent
  • Inline code and code blocks
  • Tables with striped rows
  • Images with responsive sizing
  • Horizontal rules

3. MarkdownEditorComponent app-markdown-editor

Full-featured markdown editor with live preview, toolbar actions, and form integration.

Features

  • Full toolbar with 17 actions organized in 7 groups
  • Side-by-side markdown and preview modes
  • Fullscreen editing mode
  • Keyboard shortcuts (Ctrl+B, I, K, Z, Y)
  • Undo/Redo with 50-action history
  • Character, word, and line count
  • Form integration via ControlValueAccessor
  • Disabled state support
  • Real-time preview rendering

Toolbar Actions

  • Text Formatting: Bold, Italic, Strikethrough
  • Headings: H1, H2, H3
  • Links & Media: Link, Image, Code, Code Block
  • Lists & Quotes: Quote, Unordered List, Ordered List, Table
  • Other: Horizontal Rule, Undo, Redo, Preview, Fullscreen

Usage in Forms

// In component TypeScript
form = this.fb.group({
  content: ['']
});

<!-- In template -->
<app-markdown-editor 
  formControlName="content"
  [placeholder]="'Write your article content...'"
  [minHeight]="'500px'">
</app-markdown-editor>

Usage as Standalone

<app-markdown-editor 
  [(ngModel)]="content"
  (contentChange)="onContentChange($event)">
</app-markdown-editor>
