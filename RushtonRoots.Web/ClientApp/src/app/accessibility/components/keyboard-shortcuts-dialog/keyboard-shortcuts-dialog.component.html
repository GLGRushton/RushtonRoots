General Tips

  • Use Tab to move forward through interactive elements
  • Use Shift + Tab to move backward through interactive elements
  • Use Enter or Space to activate buttons and links
  • Use arrow keys to navigate within menus and dropdowns
  • Press Esc to close dialogs and menus
