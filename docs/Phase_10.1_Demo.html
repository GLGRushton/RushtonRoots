üé® Global Accessibility Styles

Comprehensive SCSS with WCAG 2.1 AA compliance:

  • :focus-visible support with 3px yellow outline (#ffeb3b)
  • Enhanced focus for buttons, links, form fields, cards
  • Material component focus overrides
  • Screen reader only content (.sr-only)
  • High contrast mode (@media prefers-contrast: high)
  • Reduced motion (@media prefers-reduced-motion: reduce)
  • Dark mode (@media prefers-color-scheme: dark)
  • Color contrast 4.5:1 (normal), 3:1 (large text)
  • Touch targets 44x44px minimum (WCAG AAA)
  • ARIA live region styling
  • Form accessibility (required fields, errors)
  • Modal/dialog accessibility
  • Print accessibility optimizations
